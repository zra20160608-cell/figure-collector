# 手办信息收集网站 (Figure Collector)

一个聚合国内主流手办平台信息的网站，为手办爱好者提供便捷的查询、价格监控和上新提醒服务。

**启动日期**: 2026-03-27

## 🎯 项目目标

- 聚合国内主流手办平台信息（B站会员购、淘宝、京东等）
- 提供价格监控和历史趋势分析
- 上新提醒功能，不错过心仪手办
- 建立手办爱好者交流社区
