# 手办信息收集网站 - 产品需求文档 (PRD)

**项目代号**: figure-collector  
**版本**: v0.1 (草案)  
**创建日期**: 2026-03-27  
**创建人**: 项目经理 (PM-Agent)  

---

## 1. 项目背景

用户（尧）需要一个专注于国内手办信息收集的网站，聚合多个平台的手办数据，为用户提供便捷的查询、监控和提醒服务。

## 2. 产品目标

- 聚合国内主流手办平台信息
- 提供价格监控和上新提醒
- 建立手办爱好者社区
- 打造权威的手办数据库
