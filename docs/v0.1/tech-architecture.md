## 2. 技术栈选型

### 2.1 后端技术栈

| 层级 | 技术选型 | 理由 |
|------|----------|------|
| **API框架** | Node.js + Express 或 Python + FastAPI | 轻量、高性能、易维护 |
| **数据库** | PostgreSQL 14+ | 关系型、JSON支持、成熟稳定 |
| **缓存** | Redis 7+ | 高性能、支持多种数据结构 |
| **ORM** | Prisma (TypeScript) 或 SQLAlchemy (Python) | 类型安全、开发效率高 |
| **认证** | JWT | 无状态、易扩展 |
| **任务队列** | Redis + Bull / Celery | 异步任务、定时爬虫 |
| **API文档** | Swagger/OpenAPI | 自动生成、易于调试 |

### 2.2 爬虫技术栈

| 层级 | 技术选型 | 理由 |
|------|----------|------|
| **基础爬虫** | Python + Requests + BeautifulSoup | 简单页面、快速开发 |
| **动态页面** | Playwright / Selenium | JavaScript渲染页面 |
| **爬虫框架** | Scrapy (可选) | 大规模爬取、数据管道 |
| **反爬应对** | 代理池 + Cookie池 + UA轮换 | 绕过反爬机制 |

### 2.3 前端技术栈

| 层级 | 技术选型 | 理由 |
|------|----------|------|
| **框架** | Next.js 14 (App Router) | SSR/CSR兼顾、SEO友好 |
| **语言** | TypeScript 5+ | 类型安全 |
| **UI库** | Tailwind CSS + 自建组件 | 轻量、高度定制 |
| **状态管理** | Zustand | 轻量、简洁 |
| **服务端状态** | TanStack Query (React Query) | 数据获取、缓存 |
| **HTTP客户端** | Axios | 功能丰富、易配置 |
| **路由** | Next.js App Router | 文件路由、布局系统 |

### 2.4 部署技术栈

| 层级 | 技术选型 |
|------|----------|
| **容器** | Docker + Docker Compose |
| ** orchestration** | Kubernetes (可选，生产环境) |
| **CI/CD** | GitHub Actions |
| **CDN** | CloudFlare / 阿里云OSS |
| **监控** | Prometheus + Grafana |
| **日志** | ELK Stack / Loki |

## 4. 数据流设计

### 4.1 爬虫数据流

```
1. Scheduler 触发任务
   ↓
2. Worker 发起 HTTP 请求
   ↓
3. 获取 HTML/JSON
   ↓
4. Parser 解析数据
   ↓
5. Data Cleaner 清洗数据
   - 去除HTML标签
   - 标准化价格
   - 标准化时间
   - 映射厂商/品牌
   ↓
6. Repository 存储到数据库
   ↓
7. 记录价格历史
   ↓
8. 清理缓存 (Redis)
   ↓
9. 任务完成记录
```

### 4.2 前端请求数据流

```
1. 用户操作 (点击/输入)
   ↓
2. React Query 发起请求
   ↓
3. Axios 发送 HTTP
   ↓
4. Middleware 处理
   - 验证Token
   - 限流检查
   ↓
5. Controller 接收
   ↓
6. Service 业务处理
   ↓
7. Repository 查询数据库
   ↓
8. 返回数据
   ↓
9. React Query 缓存
   ↓
10. UI 更新
```

## 5. 安全设计

### 5.1 认证授权

| 场景 | 方案 |
|------|------|
| 用户登录 | JWT (Access Token) |
| Token刷新 | Refresh Token |
| 接口权限 | JWT Claims + Middleware |
| 管理员 | role字段区分 |

### 5.2 安全措施

| 措施 | 实现 |
|------|------|
| SQL注入 | ORM参数化查询 |
| XSS | React自动转义 |
| CSRF | SameSite Cookie |
| 密码存储 | bcrypt哈希 |
| API限流 | Redis计数器 |
| 请求日志 | Morgan |
