# 前端详细需求文档 (REQ-Frontend)

**文档版本**: 1.0  
**创建时间**: 2026-03-31  
**项目**: figure-collector  
**基于**: PRD v0.2、REQ-backend-api.md

---

## 1. 概述

### 1.1 文档目标
本文档定义手办信息收集网站的**前端详细需求**，精确到每个页面的UI组件、功能点和交互逻辑。

### 1.2 技术栈建议
- **框架**: React 18+ / Next.js 14+
- **语言**: TypeScript 5+
- **样式**: Tailwind CSS / CSS Modules
- **状态管理**: Zustand / Redux Toolkit
- **UI组件库**: Ant Design / Material UI (可定制)
- **HTTP客户端**: Axios / React Query
- **路由**: React Router v6 / Next.js App Router

## 2. 设计规范

### 2.1 设计风格
- **整体风格**: 简约现代、清新二次元
- **主题**: 浅色主题为主，支持深色模式
- **圆角**: 8px (卡片), 4px (按钮), 12px (弹窗)
- **阴影**: 柔和投影 `0 2px 8px rgba(0,0,0,0.08)`

### 2.2 色彩系统

#### 主色调
| 色彩 | 色值 | 用途 |
|------|------|------|
| 主色 | `#6366F1` (Indigo-500) | 按钮、链接、高亮 |
| 主色浅 | `#818CF8` (Indigo-400) | Hover状态 |
| 主色深 | `#4F46E5` (Indigo-600) | Active状态 |

#### 功能色
| 色彩 | 色值 | 用途 |
|------|------|------|
| 成功 | `#10B981` | 上新、发售中 |
| 警告 | `#F59E0B` | 预定中、价格变动 |
| 错误 | `#EF4444` | 售罄、错误状态 |
| 信息 | `#3B82F6` | 制作决定 |

#### 中性色
| 名称 | 色值 | 用途 |
|------|------|------|
| 背景 | `#F9FAFB` | 页面背景 |
| 卡片 | `#FFFFFF` | 卡片背景 |
| 文字主 | `#111827` | 标题、重要文字 |
| 文字次 | `#6B7280` | 描述、次要信息 |
| 边框 | `#E5E7EB` | 分隔线、边框 |

### 2.3 字体系统
```
字体家族:
- 中文: "PingFang SC", "Microsoft YaHei", sans-serif
- 英文/数字: "Inter", "SF Pro Display", sans-serif
- 代码: "JetBrains Mono", monospace

字号:
- H1: 32px / 2rem / font-weight: 700
- H2: 24px / 1.5rem / font-weight: 600
- H3: 20px / 1.25rem / font-weight: 600
- Body: 16px / 1rem / font-weight: 400
- Small: 14px / 0.875rem / font-weight: 400
- Caption: 12px / 0.75rem / font-weight: 400
```

### 2.4 间距系统
```
基础单位: 4px
间距:
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
```

### 2.5 响应式断点
```
移动端: < 640px (sm)
平板: 640px - 1024px (md/lg)
桌面: > 1024px (xl)
大屏: > 1280px (2xl)
```

## 5. API对接规范

### 5.1 接口列表

| 页面 | 接口 | 方法 |
|------|------|------|
| 首页统计 | GET /api/v1/stats/home | 首页加载 |
| 最新情报 | GET /api/v1/news?limit=5 | 首页 |
| 推荐手办 | GET /api/v1/figures?limit=8 | 首页 |
| 手办列表 | GET /api/v1/figures | 列表页 |
| 手办详情 | GET /api/v1/figures/:id | 详情页 |
| 厂商列表 | GET /api/v1/manufacturers | 筛选侧边 |
| 品牌列表 | GET /api/v1/brands | 筛选侧边 |
| 情报列表 | GET /api/v1/news | 情报页 |
| 登录 | POST /api/v1/auth/login | 用户中心 |
| 收藏列表 | GET /api/v1/user/favorites | 用户中心 |
| 添加收藏 | POST /api/v1/user/favorites | 详情页 |

### 5.2 请求/响应规范

**通用响应格式**:
```typescript
interface ApiResponse<T> {
  code: number;      // 200=成功
  message: string;  // 提示信息
  data: T;          // 业务数据
}
```

**分页响应格式**:
```typescript
interface PaginatedResponse<T> {
  items: T[];
  pagination: {
    page: number;
    limit: number;
    total: number;
    total_pages: number;
  };
}
```
