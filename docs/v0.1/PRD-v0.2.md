## 1. 项目背景与目标

### 1.1 问题陈述
手办市场信息分散在多个平台，玩家需要：
- 频繁访问多个网站查看新品信息
- 手动对比不同平台的价格
- 难以追踪预售开始/截止时间
- 容易错过限量发售

### 1.2 项目目标
构建一个聚合多平台手办信息的网站，实现：
- **信息聚合**: 自动抓取多平台手办数据
- **价格监控**: 追踪价格变化，提供购买建议
- **上新提醒**: 新品发布及时通知
- **数据可视化**: 价格趋势、热门商品排行

## 3. 用户画像

### 3.1 核心用户
**手办收藏者** (占比: 60%)
- 年龄: 20-35岁
- 消费: 年消费5000-50000元
- 需求: 追踪心仪手办的发售信息、比价购买

### 3.2 次要用户
**二手交易者** (占比: 25%)

- 需求: 了解手办入门知识、推荐商品

## 4. 功能需求

### 4.1 核心功能

#### FR-001 手办信息聚合展示
**优先级**: P0  
**描述**: 展示从多平台抓取的手办基础信息  
**数据来源**: Hpoi, GSC  
**展示字段**:
- 商品名称
- 厂商
- 价格
- 出荷时间
- 商品图片
- 来源平台

**界面原型**:
```
┌─────────────────────────────────────┐
│ [筛选栏] 厂商 | 价格范围 | 出荷时间   │
├─────────────────────────────────────┤
│ ┌────────┐ ┌────────┐ ┌────────┐  │
│ │ 图片    │ │ 图片    │ │ 图片    │  │
│ │ 名称    │ │ 名称    │ │ 名称    │  │
│ │ ￥2,780 │ │ ￥3,900 │ │ ￥5,500 │  │
│ │ INART  │ │ GSC    │ │ GSC    │  │
│ │ 2027年 │ │ 2026年 │ │ 2026年 │  │
│ └────────┘ └────────┘ └────────┘  │
└─────────────────────────────────────┘
```

#### FR-002 新品情报监控
**优先级**: P0  
**描述**: 展示最新手办情报  
**情报类型**:
- 制作决定
- 官图更新
- 开订通知
- 出荷提醒

**数据来源**: Hpoi最新情报板块

#### FR-003 价格追踪
**优先级**: P1  
**描述**: 追踪手办价格变化历史  
**功能**:
- 价格曲线图
- 最低价提醒
- 多平台比价

#### FR-004 上新提醒
**优先级**: P1  
**描述**: 用户订阅特定厂商/系列的新品通知  
**推送方式**:
- 站内通知
- 邮件通知 (可选)
- 浏览器推送 (可选)

## 5. 数据模型

### 5.1 手办商品 (Figure)
```json
{
  "id": "string",           // 内部ID
  "source_id": "string",    // 源平台ID
  "source_platform": "enum", // hpoi | gsc | ...
  "name": "string",         // 商品名称
  "name_jp": "string",      // 日文名称
  "manufacturer": "string", // 厂商
  "brand": "string",        // 品牌线
  "series": "string",       // 系列/作品
  "character": "string",    // 角色名
  "price_cny": "number",    // 人民币价格
  "price_jpy": "number",    // 日元价格
  "price_usd": "number",    // 美元价格
  "release_date": "date",   // 出荷日期
  "status": "enum",         // 预定中 | 已出荷 | 制作决定
  "info_type": "enum",      // 情报类型
  "images": ["string"],     // 图片URL列表
  "description": "string",  // 描述
  "accessories": ["string"],// 配件清单
  "urls": {                 // 各平台链接
    "hpoi": "string",
    "gsc": "string",
    "official": "string"
  },
  "created_at": "datetime",
  "updated_at": "datetime"
}
```

### 5.2 价格记录 (PriceHistory)
```json
{
  "id": "string",
  "figure_id": "string",
  "price": "number",
  "currency": "string",
  "source": "string",
  "recorded_at": "datetime"
}
```

### 5.3 情报记录 (News)
```json
{
  "id": "string",
  "title": "string",
  "type": "enum",        // 制作决定 | 预定 | 出荷 | 官图更新
  "figure_id": "string",
  "content": "string",
  "source_url": "string",
  "posted_at": "datetime",
  "created_at": "datetime"
}
```

### 6.2 推荐技术栈
- **爬虫**: Python + Scrapy / Playwright
